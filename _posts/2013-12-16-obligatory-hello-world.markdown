There is an obligatory _Hello World_ in these situations, so let it be.

{% highlight javascript %}
(function(what, who){
   return what + ", " + who;
})("Hello", "World");
//=> prints 'Hello, World' to STDOUT.
{% endhighlight %}

Here we go ;)
